早

杂谈

没怎么睡又早起的一天，很像回到了那些令人怀念的以前，哈哈。两点多的时候看了一会儿大护法，和当初在电影院看的感觉完全不一样，看到了更多的细节，以后再聊吧。那么聊聊最近。

很早之前就想建一个自己的网站，对，只是想想而已。最近突然觉得人就活这么一次，凡事都停留在想的阶段，那就太对不起自己了。所以，开始去做出一些改变，去把一些想法付诸行动。We only live once, make it count.

在这个自媒体兴盛的后博客时代，可能很少有人去使用个人网站去记录所见所思所想吧，也不能说很少，只能说比例占得很少吧。我从来没用过博客，哪怕是十年前博客很火的那个时代。日记倒是写过好几本，偶尔翻看自己的日记，总有种MDZZ的感觉，可能还是太随意了吧。更不必说各种社交网站上随性发布的文字或图片。最近浏览了很多人的个人网站（包括一个已经去世的人的），想着自己以后能给这个世界留下点什么。

嗯…到了该出门的时候了，以后再续吧。
